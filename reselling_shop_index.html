Schritt-für-Schritt: Webseite kostenlos online stellen

Folge diesen einfachen Schritten:

  1. Gehe auf GitHub.com und erstelle ein kostenloses Konto.
  2. Klicke oben rechts auf das + Symbol → wähle New repository.
  3. Gib einen Namen ein, z.B. reselling-shop, und klicke Create repository.
  4. Klicke auf Add file → Upload files und lade die Datei index.html hoch, dann auf Commit changes.
  5. Gehe in die Repository-Einstellungen → Pages.
  6. Wähle Branch main und Ordner /root, klicke auf "Save".
  7. Nach kurzer Zeit ist die Webseite online:
    https://dein-github-name.github.io/reselling-shop

Fertig! Deine Webseite ist jetzt kostenlos live.
